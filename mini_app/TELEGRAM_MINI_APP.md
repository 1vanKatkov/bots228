# Подключение Mini App к Telegram-боту

Ваше приложение развёрнуто по адресу (подставьте свой URL с Render):  
**https://bots228.onrender.com**

---

## Способ 1: Кнопка меню бота (рекомендуется)

Пользователь видит кнопку слева от поля ввода — по нажатию открывается Mini App.

1. Откройте **@BotFather** в Telegram.
2. Отправьте команду: **`/setmenubutton`**.
3. Выберите бота, к которому привязать приложение (например, нумерология или отдельный бот для Mini App).
4. Когда спросят тип кнопки — выберите **Configure menu button** / настройку кнопки меню.
5. Укажите:
   - **URL:** `https://bots228.onrender.com` (или ваш URL с Render).
   - **Текст кнопки:** например, `Открыть приложение` или `Мистические услуги`.
6. Готово. В чате с ботом слева от поля ввода появится кнопка, открывающая ваше приложение.

---

## Способ 2: Кнопка внутри бота

В коде бота добавлена кнопка «Открыть приложение» в главное меню (нумерология).  
Она открывает Mini App по нажатию.  
URL приложения задаётся переменной окружения **`MINI_APP_URL`** (например в `.env` бота):

```env
MINI_APP_URL=https://bots228.onrender.com
```

Если не задать — по умолчанию используется `https://bots228.onrender.com`.

---

## Проверка

1. Откройте бота в Telegram.
2. **Способ 1:** нажмите кнопку меню (слева от поля ввода).
3. **Способ 2:** нажмите «Открыть приложение» в меню бота.
4. Должно открыться ваше приложение (главная с нумерологией, сонником, совместимостью). Баланс подтягивается по `telegram_id` из Telegram Web App.

---

## Важно

- Сайт должен работать по **HTTPS** (Render даёт его по умолчанию).
- В приложении уже подключён `telegram-web-app.js` и чтение `initData.user.id` для запросов к API — менять ничего не нужно.
